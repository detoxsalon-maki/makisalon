# 02. 受け取り手順（マキ側）

> **このマニュアルは、パソコン初心者でも分かるように作っています。**
> 黒い画面（コマンドプロンプト）は基本使いません。
> 所要時間: 30〜60分

---

## このマニュアルを読む前に

ハリーからの招待を受けたら、まずこれを読んで、上から順番に進めてください。
分からないところは遠慮なくハリーにスクショ付きで質問してください。

---

## 全体の流れ（5ステップ）

```
Step 1: GitHub アカウントを作る（無料・5分）
   ↓
Step 2: ハリーからの招待メールを承認する（2分）
   ↓
Step 3: GitHub Desktop アプリを入れる（10分）
   ↓
Step 4: LPデータを自分のパソコンに取り込む（5分）
   ↓
Step 5: VS Code アプリを入れて、ファイルを開く（10分）
```

これで「自分のパソコンでLPを編集できる状態」になります。
編集の仕方は `03_日々の更新フロー_マキ側.md` で説明します。

---

## Step 1: GitHub アカウントを作る

GitHub（ギットハブ）は、世界中のプログラマが使っているコード保管庫サービスです。**完全無料**で使えます。

### 1-1. アカウント作成ページを開く

👉 https://github.com/signup

### 1-2. 入力項目

1. **Email**: 普段使うメールアドレスを入力
2. **Password**: 15文字以上、または8文字以上で数字と小文字を含むパスワード（忘れないようにメモ）
3. **Username**: マキさんが使いたい名前を英数字で（例: `maki-salon-2026`, `maki-makisalon` など）
   - **このユーザー名をハリーに伝えてください**（招待に必要）

   `[📸スクショ位置: signup ページの入力画面]`

### 1-3. メール認証

入力したメールアドレスに「Please verify your email」というメールが届きます。
メール内のリンクをクリックして認証してください。

これでGitHubアカウントは完成です。

---

## Step 2: ハリーからの招待を承認する

ハリーがマキさんをLPリポジトリ（=LPデータの保管箱）に招待したら、以下の手順で承認します。

### 2-1. 招待メールを確認

GitHubから「You've been added to harry-n2/makisalon」という件名のメールが届きます。

メール本文の **`View invitation`** ボタンをクリック。

`[📸スクショ位置: 招待メールの View invitation ボタン]`

### 2-2. 招待を承認

ブラウザでGitHubのページが開き、「You've been invited to collaborate on the harry-n2/makisalon repository」と表示されます。

**`Accept invitation`** ボタンをクリック。

`[📸スクショ位置: Accept invitation ボタン]`

これで、マキさんは `harry-n2/makisalon` リポジトリに編集権限を持つメンバーになりました。

### 2-3. 確認

ブラウザで以下を開いて、ページが見えればOK:
👉 https://github.com/harry-n2/makisalon

ページが「404 Not Found」なら、まだ招待Accept前か、招待が届いていない可能性。ハリーに連絡してください。

---

## Step 3: GitHub Desktop アプリを入れる

「黒い画面のコマンド」を使わずに GitHub を操作するための、無料のアプリです。

### 3-1. ダウンロード

👉 https://desktop.github.com/

「**Download for Windows**」（または Mac）の青いボタンをクリック → インストーラーがダウンロードされます。

`[📸スクショ位置: GitHub Desktop ダウンロードページ]`

### 3-2. インストール

ダウンロードした `GitHubDesktopSetup.exe` をダブルクリックすると、自動でインストールが始まります。

完了すると GitHub Desktop が自動で起動します。

### 3-3. GitHubアカウントでログイン

1. GitHub Desktop の最初の画面で **`Sign in to GitHub.com`** をクリック
2. ブラウザが開いてGitHubのログイン画面が出る → Step 1 で作ったアカウントでログイン
3. 「Authorize GitHub Desktop」のページで **`Authorize desktop`** をクリック
4. ブラウザが「Open GitHub Desktop」のダイアログを出すので **`GitHub Desktop を開く`** をクリック

   `[📸スクショ位置: Authorize 画面]`

5. GitHub Desktop に戻ったら **`Continue`** → ユーザー名・メールアドレスを確認して **`Finish`**

これで GitHub Desktop がマキさんのGitHubアカウントと連携できました。

---

## Step 4: LPデータを自分のパソコンに取り込む

「clone（クローン）」と言います。LPデータを丸ごとマキさんのPCにコピーする操作です。

### 4-1. Clone repository を選ぶ

GitHub Desktop のホーム画面で:

**`Clone a repository from the Internet...`** をクリック

`[📸スクショ位置: ホーム画面の Clone ボタン]`

### 4-2. リポジトリを選択

1. ダイアログが開いたら、上部タブの **`GitHub.com`** を選択
2. リストに `harry-n2/makisalon` が表示されている → **クリックして選択**
   - 表示されない場合は、上部の検索窓に `makisalon` と入力
3. **Local path**: 保存先フォルダ（デフォルトのままでOK・例: `C:\Users\マキ\Documents\GitHub\makisalon`）
4. **`Clone`** ボタンをクリック

   `[📸スクショ位置: Clone ダイアログ]`

ダウンロードが始まり、数分で完了します。

### 4-3. 完了

「Clone successful」と表示されたら成功。
GitHub Desktop の左上に `makisalon` が表示されます。

`[📸スクショ位置: clone完了後の画面]`

---

## Step 5: VS Code アプリを入れて、ファイルを開く

VS Code（ブイエスコード）は、テキスト・コードを編集するための無料アプリです。

### 5-1. ダウンロード・インストール

👉 https://code.visualstudio.com/download

「**Windows**」（または Mac）のボタン → インストーラーダウンロード → 実行。

インストール時の選択肢は全てデフォルトでOK。

### 5-2. 日本語化（推奨）

1. VS Code を起動
2. 左サイドバーの **四角が4つ並んだアイコン（Extensions）** をクリック
3. 検索窓に `Japanese Language Pack` と入力
4. **`Japanese Language Pack for Visual Studio Code`** を選び **Install** をクリック
5. 右下の通知で「Change Language and Restart」をクリック

   `[📸スクショ位置: Extensions 検索画面]`

### 5-3. LPデータのフォルダを開く

GitHub Desktop の画面で:

1. メニューバーの **`Repository`** → **`Open in Visual Studio Code`** をクリック
   - または **`Show in Explorer`** でフォルダを開き、そのフォルダを VS Code にドラッグ＆ドロップ

   `[📸スクショ位置: Repository メニュー]`

2. VS Code が起動し、左サイドバーに `makisalon` フォルダのファイル一覧が表示される

`[📸スクショ位置: VS Code でフォルダを開いた画面]`

## トラブルシューティング

### Q. GitHub のサインアップで「Username is not available」と出る

その名前は既に他の人が使っています。別の名前にしてください（数字や `-` を入れると通りやすい）。

### Q. 招待メールが届かない

迷惑メールフォルダを確認してください。それでもなければ、GitHubにログインして右上のベルアイコン（通知）から確認できます。
ハリーに「Pending invitation のリンクを直接送ってください」と頼んでもOK。

### Q. GitHub Desktop で Clone しても表示されない

リスト一覧が長い場合、検索窓に `makisalon` と入力すると絞り込めます。
それでも見つからない場合は、Step 2 の招待 Accept が未完了の可能性。

### Q. VS Code で日本語化したのに英語に戻ってしまった

VS Code 自体を一度終了して、もう一度起動し直してください。

## 何かあったら

スクリーンショットを撮ってハリーに送ってください。
画面のどこで詰まっているか分かれば、すぐに対応できます。

**スクショの撮り方**:
- Windows: `Windows キー + Shift + S` → 範囲選択 → 自動的にクリップボードにコピー → メール/LINEに貼り付け
- Mac: `Cmd + Shift + 4` → 範囲選択 → デスクトップに保存される
